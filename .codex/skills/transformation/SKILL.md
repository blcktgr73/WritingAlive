# transformation (Codex / OpenClaw 어댑터)

이 skill 의 **정본 절차**는 이 프로젝트의 `ops/skills/transformation.md` 에 있다.

1. `ops/skills/transformation.md` 를 읽는다.
2. Load Context → Check Themes → Define Transformation → Propose 2-3 Options 를 수행한다.
3. 구현 후 `docs/transformations/TRANSFORMATIONS.md` 에 `T-YYYYMMDD-###` 기록 + 영향 AC 를 `verify-ac` 로 연결한다.

판단 기준은 그 파일이 링크하는 palab-platform transformation-centered-development 를 따른다.
