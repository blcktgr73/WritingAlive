# WriteAlive 튜토리얼: 살리고 글쓰기 직접 체험하기

> 이 튜토리얼은 WriteAlive 플러그인을 사용하여 **살리고 글쓰기**를 직접 체험해보는 실습 가이드입니다.
> 빌 에반스의 연습 철학처럼 **"전체를 막연하게 근사하지 말고, 작은 부분을 완전히 진실하게"** 접근합니다.

## 🎯 이 튜토리얼의 목표

1. **낮은 에너지로 글쓰기 시작하기** - 완벽한 개요 없이도 글을 시작할 수 있다
2. **생성적 글쓰기 경험하기** - 씨앗에서 나무가 자라듯 글이 성장한다
3. **센터 발견하기** - 글 속에서 가장 살아있는 부분을 찾아낸다
4. **전체성(Wholeness) 느끼기** - 글이 점점 더 일관되고 강해지는 것을 체감한다

---

## 📖 시나리오: "나의 연습 방법"에 대한 글 쓰기

### 배경 스토리
당신은 최근 빌 에반스의 연습 철학에 대한 영상을 봤습니다:
> "전체를 막연하게 근사하지 말고, 작은 부분을 완전히 진실하게, 완전히 현실적으로, 완전히 정확하게 해라."

이 말이 당신의 기타 연습, 글쓰기 연습, 심지어 프로그래밍 학습에도 적용된다는 걸 깨달았습니다.
이 깨달음을 글로 정리하고 싶지만, 어디서부터 시작해야 할지 막막합니다.

**전통적인 방법이라면?**
- "서론: 연습의 중요성"
- "본론 1: 빌 에반스의 철학"
- "본론 2: 내 경험에 적용"
- "결론: 앞으로의 다짐"

이렇게 구조를 다 짜고 나면 이미 지쳐서 글을 쓰지 못합니다.

**살리고 글쓰기라면?**
씨앗 하나만 있으면 됩니다. 지금 바로 시작할 수 있습니다.

## 🌱 1단계: 씨앗 모으기 (Seed Gathering)

### 1-1. 일주일간 아이디어 수집하기

WriteAlive는 당신이 이미 메모해둔 생각들을 모읍니다.

**월요일 - 출근길 지하철에서 (모바일)**
```markdown
# 2025-11-04-subway-note.md
빌 에반스 영상 봤다.
"전체를 막연하게 근사하지 말라"는 말이 충격적이었다.
나는 항상 기타 곡 전체를 대충 치려고 했었다.
#seed #연습방법
```

**화요일 - 점심시간 (모바일 음성 메모)**
```markdown
# 2025-11-05-lunch-idea.md
어제 기타 연습할 때
첫 4마디만 완벽하게 연습했더니
나머지도 쉽게 따라왔다
#idea #기타 #💡
```

**수요일 - 회의 중 떠오른 생각 (데스크탑)**
```markdown
# 2025-11-06-meeting-thought.md
코드 리뷰 받으면서 깨달았다.
작은 함수 하나를 제대로 짜는 게
전체 구조를 대충 짜는 것보다 낫다.
이것도 빌 에반스 방식이네?
#씨앗 #프로그래밍
```

**목요일 - 저녁 산책 중 (모바일 사진 + 메모)**
```markdown
# 2025-11-07-park-walk.md
![[tree-growth.jpg]]
나무는 줄기에서 가지로 자란다.
글도 이렇게 써야 하는 거 아닐까?
#seed #자연관찰
```

### 1-2. WriteAlive로 씨앗 모으기

**금요일 저녁 - 드디어 글 쓸 시간!**

1. **Obsidian 열기**
2. **Ctrl/Cmd + P** → `WriteAlive: Gather Seeds` 명령 실행
3. **태그 필터 패널 보기:**

```markdown
╔══════════════════════════════════════════════════════════╗
║  🌱 씨앗 모으기                                          ║
╠══════════════════════════════════════════════════════════╣
║                                                          ║
║  📊 태그로 필터링 (any/all) [전환: all ▼]               ║
║  ┌────────────────────────────────────────────────────┐ ║
║  │ #seed (8)  #idea (6)  #💡 (4)  #연습방법 (3)     │ ║
║  │ #기타 (2)  #프로그래밍 (2)  #자연관찰 (1)        │ ║
║  │                                                    │ ║
║  │ [태그 12개 더 보기 ▼]                             │ ║
║  └────────────────────────────────────────────────────┘ ║
║                                                          ║
║  📅 날짜 필터: [이번 주 ▼]                              ║
║                                                          ║
╚══════════════════════════════════════════════════════════╝
```

4. **태그를 클릭해서 필터링** (`#연습방법`과 `#기타` 클릭해보세요):

```markdown
🌱 발견된 씨앗들 (4개) → 필터링됨 (2개)

[활성 태그 필터: #연습방법 OR #기타]

✅ 2025-11-04 - "전체를 막연하게 근사하지 말라" #seed #연습방법
✅ 2025-11-05 - "첫 4마디만 완벽하게" #idea #기타 #💡

[필터 해제]
```

5. **"필터 해제"를 클릭**하면 모든 씨앗이 다시 보입니다:

```markdown
🌱 발견된 씨앗들 (4개)

✅ 2025-11-04 - "전체를 막연하게 근사하지 말라" #seed #연습방법
✅ 2025-11-05 - "첫 4마디만 완벽하게" #idea #기타 #💡
✅ 2025-11-06 - "작은 함수 하나를 제대로" #씨앗 #프로그래밍
✅ 2025-11-07 - "나무는 줄기에서 가지로" #seed #자연관찰
   📷 tree-growth.jpg

[모두 선택] [선택 해제]
```

**왜 대단한가?**
- **스마트 태그 필터링**: 태그를 클릭하면 즉시 씨앗이 필터링됨
- **AND/OR 모드**: "아무 태그"(OR) 또는 "모든 태그"(AND) 논리 전환 가능
- **태그 통계**: 각 태그를 가진 씨앗 개수 표시 (예: `#seed (8)`)
- **한글 태그 인식** (`#씨앗`, `#연습방법`)
- **이모지 태그** (`#💡`)
- **모바일 메모 자동 동기화**
- **사진 포함** 📷 표시
- **다국어 지원** (한글, 영어, 혼용 모두 가능)
- **세션 기억**: 다시 열 때 마지막 필터 기억

모두 선택 완료. 다음 단계로!

```
### ⭐⭐⭐ 깊이 더하기 - 구체적 경험 추가
**방향**: 기타 첫 4마디를 완벽하게 연습했을 때 무슨 일이 일어났나요?

**왜 중요한가**: 당신의 글에는 강력한 개념이 있지만, 그것을 뒷받침할 생생한 경험이 필요합니다. 독자는 감각적 세부사항과 구체적인 순간을 통해 연결됩니다.

**내용 힌트**:
- 손가락이 패턴을 "알게" 되었을 때의 신체적 감각을 묘사하세요

**왜 중요한가**: 한계와 도전을 다루는 것은 당신의 글에 깊이와 정직함을 더합니다.

**내용 힌트**:
- "완벽한 4마디" 방법이 실패하는 때는 언제인가요?
- 전체를 먼저 봐야 할 때도 있나요?
- 언제 확대하고 언제 축소해야 하는지 어떻게 아나요?

**예상 분량**: 1-2 문단

---

**비용**: $0.02 (3,247 토큰) • **제공자**: Claude 3.5 Sonnet
```

당신은 **"깊이 더하기"** 제안을 선택합니다 (가장 살아있다고 느껴져서).

### 3-2. 구체적 경험 추가

```markdown
# My Practice Method

빌 에반스는 말했다. "전체를 막연하게 근사하지 말고,
작은 부분을 완전히 진실하게 해라."
나는 이 말을 듣고 내 기타 연습을 돌아봤다.

어제 나는 새 곡을 배우려다가 좌절했었다.
3분짜리 곡 전체를 한 번에 따라치려고 했고,
한 시간이 지나도 여전히 어색하고 버벅거렸다.

그런데 빌 에반스의 말을 떠올리고 방법을 바꿨다.
딱 첫 4마디만 연습했다.
같은 4마디를 20번 반복했다.

놀라운 일이 일어났다.
20번째쯤 되니 손가락이 저절로 움직였다.
생각하지 않아도 정확한 음이 나왔다.
그제서야 나는 "완전히 진실하게" 연주한다는 게
무슨 의미인지 몸으로 이해했다.
```

**전체성(Wholeness) 점수: 7.2/10** (WriteAlive 자동 분석)

## 🔄 5단계: 반복적 개선 (Iterative Refinement)

### 5-1. 나무 비유 추가 (2차 확장)

AI 제안대로 옵션 B를 선택:

```markdown
# My Practice Method

[...앞부분 유지...]

놀라운 일이 일어났다.
20번째쯤 되니 손가락이 저절로 움직였다.
생각하지 않아도 정확한 음이 나왔다.

산책하며 본 나무가 떠올랐다.
나무는 줄기에서 가지로 자란다.
처음부터 전체 나뭇가지를 한꺼번에 뻗지 않는다.
줄기가 튼튼해지면, 가지가 자연스럽게 나온다.
각 가지가 제대로 자라면, 잎이 무성해진다.

내 연습도 그래야 한다는 걸 깨달았다.
첫 4마디라는 '줄기'가 튼튼해지자,
다음 4마디라는 '가지'는 저절로 따라왔다.
```

**전체성 점수: 8.3/10** ⬆️ (+1.1)

### 5-2. 소리 내어 읽기

WriteAlive는 전통적 글쓰기 방법도 지원합니다:

**Cmd/Ctrl + P** → `WriteAlive: Read Aloud Feedback`

```
🔊 소리 내어 읽기 분석

✅ 잘 읽히는 부분:
  • "놀라운 일이 일어났다" - 리듬감 좋음
  • "줄기 → 가지 → 잎" - 반복이 명확함

⚠️ 걸리는 부분:
  • "그제서야 나는" - 약간 어색
  → 제안: "그때 비로소"

  • "생각하지 않아도 정확한 음이" - 너무 긺
  → 제안: "생각 없이도 정확한 음이" 또는 두 문장으로 분리
```

## 💬 자주 묻는 질문

**Q: 씨앗이 너무 많으면 어떡하죠?**
A: AI가 관련성 높은 것만 제안합니다. 나머지는 다른 글의 씨앗이 됩니다.

**Q: 센터를 못 찾겠어요.**
A: AI가 도와줍니다. 그래도 확신이 안 서면 "느낌 가는 씨앗" 하나로 시작하세요.

**Q: 전체성 점수가 낮게 나와요.**
A: 괜찮습니다! 7.0 이상이면 좋은 글입니다. 9.0 이상은 탁월한 글입니다.

**Q: 전통적 구조(서론-본론-결론)와 충돌하지 않나요?**
A: 아닙니다. 살리고 글쓰기로 내용을 키우고, 나중에 Academic Writing 기법으로 다듬으면 됩니다.

**Q: 영어로도 되나요?**
A: 네! 한글, 영어, 혼용 모두 지원합니다.
